Support

To obtain support, download diagnostic information through this page.

If the downloaded zip file is less than 1/2 GB then attach it to an email message that requests help and send the message to fuse-online-tech-preview@redhat.com.

If the downloaded zip file is larger then 1/2 GB, send an email message to request help, do not attach the file but indicate that the zip file is too large to attach. Technical support will provide instructions for sending the zip file.
