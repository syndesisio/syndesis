Support

To obtain support, download diagnostic information through this page. If the downloaded zip file is less than 500 MB, attach it to an email message that requests help and send the message to fuse-online-tech-preview@redhat.com. If the downloaded zip file is larger than 500 MB, send an email message to request help, but do not attach the file, and indicate that the zip file is too large to attach. Technical support will provide instructions for sending the zip file.
