= Creating integrations
:context: create

After some planning and preparation, you are ready to create an
integration. In the {prodname} web interface, when you
click *Create Integration*, {prodname} guides you
through the procedure to create an integration.

.Prerequisites
* link:{LinkFuseOnlineIntegrationGuide}#plan_ready[Considerations for planning your integrations]
* According to the kind of integration that you want to create: 
** An understanding of the link:{LinkFuseOnlineIntegrationGuide}#workflow-overview_ready[general workflow for creating a simple integration]
** An understanding of the link:{LinkFuseOnlineIntegrationGuide}#workflow-api-providers_api-provider[general workflow for creating an API provider integration]
